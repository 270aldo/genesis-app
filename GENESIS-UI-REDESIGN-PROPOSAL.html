Estado actual

Lo que funciona, lo que falta, y dónde está el mayor impacto.

CRÍTICO

Onboarding sin alma

5 pantallas con cards oscuras y texto. Sin imágenes, sin ilustraciones, sin GENESIS guiándote. Parece un formulario, no una experiencia.

CRÍTICO

GENESIS chat escondido

El chat con GENESIS es un modal. No hay acceso prominente. GENESIS debería ser omnipresente como el core de la app.

ALTO

Zero assets visuales

Carpeta assets/ solo tiene iconos de app. No hay imágenes de ejercicios, fondos de cards, ilustraciones, ni animaciones Lottie.

ALTO

Empty states genéricos

Track muestra "0, 0, —" sin contexto. Sin ilustraciones ni motivación. Los estados vacíos son la primera impresión del usuario nuevo.

MEDIO

Sin micro-interactions

No hay animaciones de entrada, transiciones entre pantallas, ni feedback háptico. Se siente estático.

MEDIO

Densidad visual inconsistente

Home es denso y rico. Mind y Track se sienten vacíos. Fuel está bien pero le falta carácter visual.

Prioridad 2

Home Dashboard

El Home es la mejor pantalla actual. Propuestas para llevarlo al siguiente nivel.

MEJORAS
1

GENESIS card → Hero visual

El briefing de GENESIS debería tener un fondo subtle con gradiente + avatar animado. Convertirlo en el centerpiece visual del Home.

2

Daily Missions → Cards con background images

Train, Fuel, Check-in cards deberían usar ImageCard con fondos temáticos (gym, food, meditation) + overlay gradients.

3

Quick Metrics → Animated counters

Los 4 metrics (kcal, sleep, water, steps) deberían tener counters animados al entrar y micro-charts de tendencia.

4

Floating GENESIS button

FAB permanente con el icono de GENESIS (cpu) + pulse animation. Acceso instantáneo al chat desde cualquier tab.

5

Week grid → Rings de actividad

El grid de 7 días podría convertirse en mini activity rings (estilo Apple Watch) mostrando train/fuel/mind completion.

Tab Screens

Mejoras específicas para cada tab — Train, Fuel, Mind, Track.

Train

5 MEJORAS
→ Hero image card con foto real del tipo de entrenamiento (push/pull/legs)
→ Exercise rows con thumbnails de video (preview frames)
→ Animated muscle group indicator (heatmap visual del workout)
→ Rest day: ilustración motivacional + GENESIS tip

Fuel

4 MEJORAS
→ Circular progress → animated ring con gradient (no solo outline)
→ Meal cards con food icons temáticos o fotos del scanner
→ Water tracker → animated fill effect (no solo dots)
→ Macro bars → gradient fills con labels más visibles

Mind

5 MEJORAS
→ Recovery heatmap → body silhouette visual (no solo grid de texto)
→ Mood selector → emojis más grandes + animated selection
→ Meditation cards ya lucen bien → añadir duration badge + play button más prominente
→ Sleep tracker → visual timeline (no solo número)
→ Wellness score → radar chart o gauge animated

Track

6 MEJORAS
→ Empty state → ilustración motivacional + "Completa tu primer workout para ver stats"
→ Stats (0, 0, —) → animated counters que suben al cargar
→ Strength chart → line chart interactivo (touch to see value)
→ PRs → trophy cards con confetti animation al lograr uno nuevo
→ Progress photos → before/after slider comparison
→ GENESIS Insight → expandable card con AI analysis
Core Feature

GENESIS Chat

GENESIS debe ser omnipresente. No escondido en un modal — presente en toda la experiencia.

Floating Action Button

GENESIS FAB persistente en TODAS las pantallas (bottom-right, encima del tab bar). Tap → abre chat sheet. Long press → voice mode.

Contextual Suggestions

Quick action pills adaptados al contexto. En Train → "¿Sustituyo este ejercicio?". En Fuel → "¿Qué comí hoy es suficiente?". En Mind → "¿Cómo mejorar mi sueño?"

Proactive Insights

GENESIS manda "nudges" en Home: "Tu sueño bajó 2 noches seguidas. ¿Revisamos tu rutina nocturna?" — como un coach real.

Sistema

Sistema Visual

Assets, animaciones e imágenes que necesitamos crear/integrar.

Image Assets Necesarios

Exercise thumbnails

~50 imágenes de ejercicios con overlay oscuro. Fuente: Unsplash fitness + AI generation.

Onboarding illustrations

5 ilustraciones (goal/level/schedule/body/review) estilo minimal dark. SVG preferred.

Mission card backgrounds

Train/Fuel/Mind/Track hero images. Lifestyle photography + gradient overlays.

Meditation scene backgrounds

3-5 ambient scenes (montaña, océano, bosque). Ya hay en Mind pero expandir.

Empty state illustrations

4 ilustraciones SVG para estados vacíos de cada tab.

Animaciones a Implementar

Screen transitions (Reanimated 4)

Fade + slide para tab switches. SharedElement para card → detail transitions.

Animated counters

Números que "cuentan" al entrar en pantalla. Para metrics, stats, progress %.

Progress ring animation

Circular progress con fill animation (Fuel calories, Season progress).

Card entrance stagger

Cards que entran escalonadas (200ms delay entre cada una) con fadeInUp.

Haptic feedback

expo-haptics: light para selección, medium para completar set, heavy para PR.

Skeleton loaders

Ya existe ShimmerEffect.tsx — asegurar que se usa en TODAS las pantallas durante carga.
