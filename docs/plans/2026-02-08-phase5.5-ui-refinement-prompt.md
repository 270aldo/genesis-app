## CONTEXTO

Soy Aldo, CEO de NGX. Estoy construyendo la app GENESIS (Expo SDK 54 + React Native + NativeWind v4). La app tiene 20 pantallas, 53 componentes y ~51K líneas de código. Phases 1-5 están completas (foundation, tabs, screens, polish, backend scaffold).

Antes de avanzar a Phase 6 (Functional MVP), necesito refinar el diseño de varias pantallas. La app funciona, pero el UI necesita ser premium — no prototype.

## DESIGN SYSTEM

- **Color primario:** `#6D00FF` (NGX violet) — este es el branding oficial, NO el rosa/morado que hay actualmente
- **Background gradient:** `['#0D0D2B', '#1A0A30']`
- **Surface:** `bg-[#14121aB3]` con `border-[#FFFFFF14]`
- **Glassmorphism:** backdrop-blur, transparencies, multiple shadows, subtle gradients
- **Fonts:** Inter (body), JetBrains Mono (data/monospace)
- **Icons:** Lucide React Native exclusively (no emojis)
- **Style:** 80% effects + 20% colors. Depth over flat, tactile over generic.

## LAS 7 INCONFORMIDADES (SCOPE)

### 1. HOME — Genesis Briefing Card
**Archivo:** `app/(tabs)/index.tsx` + `components/home/DailyBriefing.tsx`
**Problema:** La card principal donde dice "GENESIS — Buenas tardes..." es plana y no resalta. Es lo primero que ve el usuario.
**Lo que quiero:** Fondo con estilo liquid glass distinto al resto. Que se sienta como una "transmisión" de GENESIS. Efecto premium que haga que esta card sea el focal point del Home.

### 2. GENESIS CHAT — Rediseño completo
**Archivo:** `app/(screens)/genesis-chat.tsx` + `components/genesis/GenesisChat.tsx` + componentes relacionados
**Problema:** El chat es "totalmente horrible". Necesita refinamiento profundo en layout, burbujas, input area, tipografía, spacing, y feel general.
**Lo que quiero:** Chat premium tipo app de lujo. Burbujas con glassmorphism sutil para GENESIS, limpias para el usuario. Input area pulida. Quick actions bien diseñadas. Typing indicator elegante. Que se sienta como hablar con un sistema avanzado, no con un chatbot genérico.

### 3. COLOR BRANDING — Violet #6D00FF everywhere
**Archivos:** `constants/theme.ts`, `constants/Colors.ts`, `app/(tabs)/_layout.tsx`, y todos los componentes que usen el morado/rosa actual
**Problema:** La barra de navegación y otros elementos usan un morado tipo rosa que NO es el branding. El color correcto es `#6D00FF`.
**Lo que quiero:** Auditoría completa de color. Reemplazar todo el rosa/morado incorrecto por `#6D00FF` y sus variantes (con opacidades para hover states, disabled, etc).

### 4. EXERCISE DETAIL — Refinamiento del log de ejercicio
**Archivo:** `app/(screens)/exercise-detail.tsx` + `components/training/ExerciseForm.tsx`
**Problema:** La pantalla de logging de ejercicio (ej: Bench Press dentro de Push Day) necesita mejor acomodo. El flujo de Push Day es GENIAL, pero al entrar al detalle del ejercicio falta refinamiento.
**Lo que quiero:** Mejor layout de sets/reps/weight, spacing correcto, visual hierarchy clara, que se sienta como un tracking tool profesional.

### 5. EXERCISE LIBRARY — Mejor organización
**Archivo:** `app/(screens)/exercise-library.tsx` + `components/training/ExerciseLibraryCard.tsx`
**Problema:** La librería está buena pero necesita mejor organización visual — layout, filtros, cards, spacing.
**Lo que quiero:** Grid o list view más refinado, filtros más accesibles, cards con mejor información visual, que se sienta como navegar un catálogo premium.

### 6. FUEL PAGE — Scroll roto
**Archivo:** `app/(tabs)/fuel.tsx`
**Problema:** La página no tiene scroll. El contenido está cortado y no se puede acceder a todo.
**Lo que quiero:** Scroll funcional + revisar el layout general para que el contenido fluya bien.

### 7. MIND PAGE — Scroll roto + mejoras
**Archivo:** `app/(tabs)/mind.tsx`
**Problema:** Me gusta el diseño general pero no tiene scroll, y la sección necesita mejoras.
**Lo que quiero:** Scroll funcional + refinamiento de la sección para que se sienta más completa y pulida.
