## Heuristics (H1-H10)

| ID | Heuristic | Description |
|----|-----------|-------------|
| H1 | Visibility of System Status | App keeps user informed about what's happening |
| H2 | Match Between System and Real World | Uses language/concepts familiar to fitness users |
| H3 | User Control and Freedom | Easy to undo, go back, exit flows |
| H4 | Consistency and Standards | UI patterns are consistent across screens |
| H5 | Error Prevention | Design prevents errors before they happen |
| H6 | Recognition Rather Than Recall | Information visible, not memorized |
| H7 | Flexibility and Efficiency | Shortcuts for experienced users, streamlined flows |
| H8 | Aesthetic and Minimalist Design | No clutter, focused content, premium feel |
| H9 | Help Users Recognize and Recover from Errors | Clear error messages with solutions |
| H10 | Help and Documentation | Guidance available when needed |
